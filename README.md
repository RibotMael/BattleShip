# Battleship Game

Bienvenue sur **Battleship**, un jeu de bataille navale moderne jouable directement en ligne ! 

👉 **Jouer maintenant :** https://battle-ship-seven.vercel.app/

---

## À propos du projet

Battleship est une adaptation du célèbre jeu de bataille navale, développée avec une approche moderne et multijoueur. Le projet est **toujours en cours de développement**.
Projet développé dans le cadre de mon bts .

---

## Fonctionnalités actuelles

* **Mode 1 vs 1** : Affrontez un autre joueur en duel classique
* **Mode Battle Royale** : Plusieurs joueurs s’affrontent jusqu’au dernier survivant
* **Jeu en ligne** : Rejoignez des parties facilement
* **Responsive** : Jouable sur **PC et mobile**

---

## Technologies utilisées

* Frontend : Vue.js
* Backend : Node.js / Express
* Base de données : MySQL
* Temps réel : WebSockets (Socket.io)
* Déploiement : Vercel
